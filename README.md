# Submission marker #

Moodle plugin to mark completed exercises

![Submission marker](https://raw.githubusercontent.com/UniversityOfHelsinkiCS/submarker/master/submarker.png)

## Installation ##

Pull this repository to moodle/mod/assign/submission
